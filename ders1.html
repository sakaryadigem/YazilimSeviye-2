Ders 1
HTML'e Giriş

Web sayfalarının temel yapı taşlarını öğreniyoruz.

Konu 1

HTML Nedir?

HTML (HyperText Markup Language), web sayfalarının iskeletini oluşturan bir işaretleme dilidir. Programlama dili değildir; tarayıcıya içeriğin nasıl yapılandırılacağını söyler.

Bir web sayfası düşünüldüğünde HTML içeriği, CSS görünümü, JavaScript davranışı yönetir. Biz bu derste HTML'in temellerini öğreneceğiz.

Konu 2

Temel Sayfa İskeleti

Her HTML belgesi aşağıdaki temel yapıyla başlar. Bu yapı tarayıcıya "Bu bir HTML5 belgesidir ve Türkçe karakterler içerir" der. 

<!-- Tarayıcıya: "Bu bir HTML5 belgesidir." -->
<!DOCTYPE html>
<html lang="tr">

  <head>
    <!-- Sayfayla ilgili bilgiler — kullanıcı görmez -->
    <meta charset="UTF-8" />
    <title>Benim Sayfam</title>
    <link href="style.css" rel="stylesheet" />
  </head>

  <body>
    <!-- Görünen içerik buraya gelir -->
    Merhaba Dünya!
  </body>

</html>
⚠️

Dikkat: <head> ile <body> birbirine karıştırılmamalı. Başlık, stil, meta bilgileri head'e; görünen her şey body'ye yazılır.

💡

Unutma: Her web sayfası aslında bir metin dosyasıdır. Uzantısı .html ya da .htm olur ve herhangi bir metin editörüyle açılıp düzenlenebilir.

Konu 3

Etiketler (Tags) Nasıl Çalışır?

HTML etiketleri açılış ve kapanış çiftlerinden oluşur. Bazı etiketler kendi kendine kapanır (self-closing). 

<!-- Açılış + içerik + kapanış -->
<p>Bu bir paragraftır.</p>

<!-- Kendi kendine kapanan etiket -->
<hr />   <!-- yatay çizgi -->
<br />   <!-- satır sonu -->
<img src="resim.jpg" alt="Açıklama" />

<!-- Özellik (attribute) kullanımı -->
<a href="https://orneksite.com" target="_blank">
  Bağlantıya tıkla
</a>
  • HTML
  • CSS
  • JavaScript
Etiket Ne İşe Yarar? Örnek
<h1> … <h6> Başlık (h1 en büyük, h6 en küçük) Sayfa başlığı, bölüm adı
<p> Paragraf Normal metin bloğu
<a> Bağlantı (link) Başka sayfaya gitme
<img> Resim Görsel ekleme
<ul> / <ol> / <li> Liste (sırasız / sıralı / öğe) Maddeli listeler
<table> Tablo Veri tabloları
<hr> Yatay çizgi Bölüm ayırıcı
<strong> Kalın (önemli) metin Vurgu yapmak için
<em> İtalik (vurgulu) metin Özel terimler
Konu 4

Bağlantılar ve Navigasyon

<a> etiketi (anchor = çapa) başka sayfalara, dosyalara ya da aynı sayfanın farklı bölümlerine bağlantı oluşturur. En önemli özelliği href'tir. 

<!-- Yeni sekmede açma -->
<a href="tablo.htm" target="_blank">İlk Menü</a>

<!-- Aynı sekmede açma (varsayılan) -->
<a href="hakkimizda.htm">Hakkımızda</a>

<!-- Dış siteye bağlantı -->
<a href="https://www.google.com" target="_blank">
  Google'a git
</a>

Bu dersin orijinal menüsü şöyle görünür:

İlk Menü İkinci Menü Üçüncü Menü Dördüncü Menü
💡

target="_blank" bağlantıyı yeni sekmede açar. Hiç yazmazsan aynı sekmede açılır.

Konu 5

Listeler

HTML'de iki tür liste vardır: sırasız (madde imli) ve sıralı (numaralı).

<!-- Sırasız liste -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Sıralı liste -->
<ol>
  <li>Önce HTML öğren</li>
  <li>Sonra CSS öğren</li>
  <li>Ardından JavaScript</li>
</ol>
Konu 6

Resim Ekleme

Resimler <img> etiketiyle eklenir. Kapanış etiketi yoktur. İki zorunlu özellik vardır: src (kaynak dosya) ve alt (alternatif metin). 

<img
  src="foto.jpg"
  alt="Güzel bir manzara fotoğrafı"
  width="400"
  height="300"
/>
⚠️

alt özelliğini boş bırakma! Ekran okuyucular görme engelli kullanıcılara resmi alt metniyle tanıtır. Aynı zamanda resim yüklenemediğinde bu metin gösterilir.

Konu 7

CSS ile Stil Vermek

HTML sayfanın içeriğini, CSS ise görünümünü belirler. CSS'i sayfaya bağlamak için <head> içine <link> etiketi eklenir. 

<head>
  <!-- Harici CSS dosyası bağlama -->
  <link
    href="style.css"
    type="text/css"
    rel="stylesheet"
  />

  <!-- Ya da doğrudan sayfa içine -->
  <style>
    body {
      background-color: #f0f0f0;
      font-family: Arial, sans-serif;
    }
  </style>
</head>
Alıştırma

Bu Dersin Alıştırması 🎯

Aşağıdakileri yaparak kendi HTML sayfanı oluştur:

  • Bir <h1> başlığı ekle (adın olsun)
  • En az 2 paragraf <p> yaz
  • <ul> ile favori 3 şeyini listele
  • Bir <a href> bağlantısı koy
  • <hr /> ile bölümleri ayır
  • Sayfayı style.css dosyasına bağla
🚀

Bonus: <table> kullanarak haftalık ders programını tabloya dökerek bir tablo.htm sayfası oluştur.
